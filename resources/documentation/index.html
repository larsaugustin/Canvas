Documentation

To move around, hold down space. You can select an element by clicking it. An element can be moved with the arrow keys and resized with those same keys while holing option/alt (shift accelerates the movement). For everything else, just start typing!

No Selection

name Change the name of the file
artboard Create a new artboard
import Import an existing file
download Download the file

Artboard

rect Create a new rectangle
triangle Create a new triangle
circle Create a new circle
text Create a new text element
color Change the background color of the artboard
x y Change the position of the artboard
width Change the width of the artboard
height Change the height of the artboard
duplicate Duplicate the artboard
clear Clear the artboard
delete Delete the artboard
export Export the artboard as an image

Rectangle

color Change color of the rectangle
x y Change the position of the rectangle
width Change the width of the rectangle
height Change the height of the rectangle
duplicate Duplicate the rectangle
delete Delete the rectangle

Triangle

color Change color of the triangle
x y Change the position of the triangle
length Change the length of the sides of the triangle
duplicate Duplicate the triangle
delete Delete the triangle

Circle

color Change color of the circle
x y Change the position of the circle
radius Change the radius of the circle
duplicate Duplicate the circle
delete Delete the circle

Text

text Change text of the element
font Change font of the text element
style Change style of the text element (for example: “bold”)
size Change size of the font of the text element
color Change color of the text element
x y Change the position of the text element
duplicate Duplicate the text element
delete Delete the text element
